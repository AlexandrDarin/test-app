
# Test App

Простое React-приложение, созданное с помощью Vite, для демонстрационных целей или в качестве стартовой точки для нового проекта.
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## Технологии

- **React** - библиотека для построения пользовательских интерфейсов
- **Vite** - инструмент для сборки и разработки
- **TypeScript** - для типобезопасности кода
- **CSS** - для стилизации компонентов

## Предварительные требования

Перед началом работы убедитесь, что на вашем компьютере установлены:
- **Node.js** (версия 16 или выше)
- **npm** (обычно устанавливается вместе с Node.js)

## Доступные скрипты
- npm run dev - запуск сервера для разработки
- npm run build - сборка проекта для продакшена
-npm run preview - предпросмотр собранной версии проекта

## Установка и запуск
**Клонируйте репозиторий** - git clone https://github.com/AlexandrDarin/test-app.git
**Перейдите в директорию проекта** - cd test-app
**Установите зависимости** - npm install
**Запустите сервер для разработки** - npm run dev
**Откройте в браузере** - После успешного запуска в терминале появится сообщение с локальным адресом (обычно http://localhost:3000). Откройте эту ссылку в вашем браузере.
  
Чтобы проверить установленные версии, выполните в терминале:
```bash
node --version
npm --version
